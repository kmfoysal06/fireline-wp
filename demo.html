How It Works

  1. Install & Activate: Add the plugin to WordPress and activate it
  2. Auto-Detection: Plugin automatically finds your theme's content area
  3. Link Interception: Internal links are caught and loaded via AJAX
  4. Progress Display: Beautiful bar shows at the top during loads
  5. Content Swap: Only the main content updates, keeping everything else
  6. URL Update: Browser URL updates, history works perfectly
